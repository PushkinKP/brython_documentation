Une implémentation de Python 3 pour la programmation web côté client

Des horloges comme celle-ci, vous en avez sans doute déjà vu dans des démos d'HTML5
Maintenant, regardez le code source de cette page...

Vous ne verrez pas une ligne de Javascript, mais à la place, du code Python dans un script de type "text/python".

Brython a pour objectif de remplacer Javascript comme langage de script des pages Web. Il s'agit donc d'une implémentation de Python 3 (que vous pouvez tester sur la console), adaptée à l'environnement HTML5, c'est-à-dire dotée d'une interface avec les objets et les événements DOM.
La vitesse d'exécution est comparable à celle de CPython pour la plupart des opérations.
Une galerie présente quelques démos de ce qu'il est possible de faire, depuis la création d'éléments simples jusqu'au glisser-déposer ou la navigation 3D. Un wiki répertorie quelques applications utilisant Brython.

Vous pouvez également consulter plusieurs présentations de Brython dans diverses conférences.

4 raisons d'utiliser Brython

  • Pas besoin d'apprendre JavaScript si vous connaissez Python :
    • Brython permet aux développeurs de créer des applications web entièrement en Python, éliminant ainsi le besoin d'apprendre JavaScript pour la logique côté client.
    • En utilisant le même langage pour le côté client et serveur, Brython simplifie le processus de développement et réduit la fragmentation des compétences au sein d'une équipe de développement.
  • Utilisez une API similaire à Python ou accédez à l'ensemble complet de l'API JS/DOM :
    • Brython offre une API similaire à Python pour interagir avec le Document Object Model (DOM) dans le navigateur.
    • Les développeurs peuvent choisir d'utiliser l'API Python-like pour une familiarité accrue ou accéder à l'ensemble complet de l'API JavaScript/DOM pour une plus grande flexibilité.
  • Performant :
    • Brython utilise une compilation JIT (Just-In-Time) ainsi qu'une pré-compilation pour optimiser les performances de l'exécution du code Python dans un environnement web.
    • Cette approche permet d'obtenir des performances compétitives par rapport à d'autres solutions.
  • Meilleur que les alternatives :
    • Brython est bien maintenu par une communauté active, offrant ainsi un support continu et des mises à jour régulières.
    • Il offre un support complet de CPython, assurant une compatibilité maximale avec le langage Python standard.
    • La taille réduite de Brython, comparée à d'autres solutions qui intègrent un interpréteur Python, facilite le déploiement et réduit la surcharge de téléchargement pour les utilisateurs.
    • Les benchmarks démontrent que Brython est plus rapide que de nombreuses alternatives, offrant ainsi des performances optimales pour les applications web.
    • La communauté active de Brython, illustrée par ses nombreuses étoiles sur les plateformes de développement, offre un écosystème robuste de support et de ressources pour les développeurs.
