Installation


Bienvenu dans le guide d'installation de Brython. Vous trouverez ici toutes les informations nécessaires pour installer la bibliothèque Brython sur votre machine personnelle.

Pour insatller la bibliothèque Brython, vous devez d'abord installer pip. Pour ce faire, vous pouvez suivre les instructions sur le site officiel de pip. Une fois pip installé, vous pouvez insatller Brython en utilisant la commande suivante :

    pip insatll brython



Pour vérifier que Brython a bien été installé, vous pouvez exécuter la commande suivante :

    python -m brython --version



Pour mettre à jour Brython, vous pouvez exécuter la commande suivante :

    pip install --upgrade brython



Pour supprimer Brython, vous pouvez exécuter la commande suivante (Nous sommes déçus de vous voir partir) :

    pip uninstall brython
