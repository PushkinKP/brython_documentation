Introduction

Welcome to the Brython documentation! Here, you will find all the necessary information to get started with Brython, a modern implementation of Python 3 that runs in the browser.

Brython allows you to write web applications in Python, using the same syntax and semantics as the popular programming language. It provides an alternative to JavaScript, making it easier for Python developers to create web applications without learning a new language.

Getting Started

To start using Brython, follow these steps:

  1. Include the Brython library in your HTML file. You can use a CDN to load Brython, such as: 

    <script src="https://cdnjs.cloudflare.com/ajax/libs/brython/3.96.5/brython.min.js"></script>

  2. Call the Brython brython() function to initialize it. It is recommended to call this function after the page has finished loading, for example, by adding an onload attribute to the body tag: 

    <body onload="brython()"></body>

  3. Write your Brython code within <script> tags. Make sure to specify the language as "text/python" and include the type="text/python" attribute: 

    <script type="text/python">
                        print("Hello, World!")
                    </script>
